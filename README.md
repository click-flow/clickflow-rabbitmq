# RabbitMQ Docker Instance
Docker files and docker-compose files for stand-alone and clustered RabbitMQ instances.
## Stand-alone
`docker-compose up`
## Example usage
### node.js:
Start the example worker listener:  
`node worker.js`

Queue up a task:  
`node tasker.js`

### ruby:
Start the example worker listener:  
`ruby worker.rb`

Queue up a task:  
`ruby tasker.rb`
